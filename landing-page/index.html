Recursos

Do JSON ao DTO.
Do scaffold ao deploy.

Duas ferramentas em uma: gerador de DTOs inteligente + scaffold de projetos com Clean Architecture e microservicos.

JSON to DTO

Envie uma URL de qualquer JSON API e receba DTOs tipados automaticamente em TypeScript, C# ou Python.

SQL to DTO

Aponte para um schema SQL e gere DTOs automaticamente. Ideal para quem migra de sistemas legacy.

Geracao de CRUD

De JSON a Controllers, Services e Repositories completos com endpoints REST (GET, POST, PUT, DELETE). Suporta .NET, Node, Python e TypeScript.

Scaffold de Projetos

Crie a estrutura de projetos backend (.NET, FastAPI, Rails, NestJS) e frontend (React, Vue, Angular, Next) em segundos.

Clean Architecture

Use --clean para gerar projetos com estrutura de microservicos, separacao por camadas e boas praticas.

Open Source

Projeto aberto, feito pela comunidade e para a comunidade. Contribua, reporte bugs e sugira melhorias.

Docker Automatico

Gere Dockerfiles otimizados e docker-compose.yml com 14+ servicos prontos (Postgres, Redis, Mongo, Nginx, etc).

Historico de Comandos

Banco SQLite local salva seu historico de comandos para referencia futura e consulta rapida.

Como Funciona

Tres comandos.
Projeto completo.

1

Instale a CLI

gem install ./dto-cli-0.1.0.gem
2

Gere DTOs + CRUD

dto-cli-dev gerar_crud --input user.json --lang dotnet --path ./src
3

Scaffold do projeto

dto-cli-dev init backend dotnet minha-api --clean
